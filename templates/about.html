About ClassCraic

ClassCraic is a real-time chat app for students and teachers. It helps users communicate in an organised way through class groups, private chats, and shared discussions.

What does ClassCraic offer?

ClassCraic is an educational chat application where teachers can create and manage class groups, and students can join those groups using a join code. This allows students to participate only in the classes they are part of. In addition to class groups, the platform includes a public chat where all users can connect for general discussions and announcements and a private chat for one to one communication among peers. The application also provides a message translation feature, allowing students to communicate more comfortably in different languages. Messages can be translated into English by just pressing a button. Users can also share files within chats, making it easier to exchange learning materials and resources.

Who can do what?

Student

  • Join classes using a code
  • Send messages in real time
  • Upload files
  • Use private chat
  • Cannot create class groups

Teacher

  • Everything a student can do
  • Create class groups (after approval)
  • Share join codes
  • Manage group members

Admin

  • Approve teachers
  • Manage users and groups
  • Full system control

How it works

1. Sign up and verify your email

2. Log in to access chat

3. Join a class using a code (students)

4. Create a class group (teachers)

5. Start chatting instantly

Features

  • Real-time messaging
  • Class groups with join codes
  • Private chat
  • File sharing (images & PDF, max 5MB)
  • Message translation

Simple example

A teacher creates a class → shares a join code → students enter it → everyone starts chatting.
